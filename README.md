# github-final-project

## Simple Interest Calculator

This repository contains a **Simple Interest Calculator** implemented as a Bash script (`simple-interest.sh`).  
The calculator allows users to compute the simple interest based on the principal amount, rate of interest, and time period.

### Features

- Calculates simple interest using the formula:  
  `SI = (Principal × Rate × Time) / 100`
- User-friendly input prompts
- Error handling for invalid input

### Usage

1. Open a terminal
2. Navigate to the repository folder
3. Run the script:

```bash
bash simple-interest.sh
